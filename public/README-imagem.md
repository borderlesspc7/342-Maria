# Imagem de fundo (login, sidebar e header)

O app usa a imagem **`/tela-fundo.png`** como fundo em:
- Tela de login
- Sidebar (menu lateral)
- Header (barra superior)

**Para a imagem aparecer:**

1. Copie o arquivo **`dist/assets/Tela Fundo.png`**
2. Cole nesta pasta (**`public/`**)
3. Renomeie para **`tela-fundo.png`** (tudo em minúsculo, com hífen)

Sem esse arquivo, o fundo mostra apenas o gradiente do tema.
